Bundled Software
-----------------------------------------------
LML includes the following components:
- Path::Class - Cross-platform path specification manipulation
- table.js - Functions for interactive Tables, Copyright (c) 2007 Matt Kruse (javascripttoolbox.com)

Required Software:
-----------------------------------------------
LML requires the following components to function:
- VMware::VIRuntime - VI Perl SDK
- SVN::Client - Subversion client functions
- DateTime::Format::Flexible - Flexibly parse strings and turn them into DateTime objects
- Config::IniFiles - A module for reading .ini-style configuration files
